Installation Guide

Installing Pegasus View

Follow these steps to add Pegasus View to Google Chrome. No technical experience required.

1

Open the Chrome Web Store listing

Open Google Chrome and search for "Pegasus View" in the Chrome Web Store, or click the link below to go directly to the listing page. The page should appear similar to the figure below.

→ Open Pegasus View on the Chrome Web Store
2

Click "Add to Chrome"

On the listing page, locate the blue "Add to Chrome" button near the top right and click it.

3

Confirm the installation

A confirmation dialog will appear. Click "Add extension" to confirm. Pegasus View will be installed and visible in your Chrome toolbar.

Operating Instructions

Using Pegasus View

Before getting started, here is an overview of every setting available to you.

Setting What it does
Font Choose from 6 dyslexia-friendly fonts (OpenDyslexic, Atkinson Hyperlegible, Lexend, Comic Neue, Verdana, Arial) or keep the site's original font.
Text Size Changes the default font size for elements such as paragraphs, lists, headings, and buttons.
Line Spacing Changes the spacing between individual lines within a paragraph. Helpful for making dense text easier to follow.
Letter Spacing Changes the horizontal space between each letter in a word. Good for anyone who finds tightly packed text hard to read.
Text Color Overrides the text color on any page with a color of your choosing.
Background Overrides the page background color. Defaults to a warm cream tone to reduce screen glare.
1

Set your preferences on first launch

As soon as Pegasus View is installed, a Welcome Page opens automatically in a new tab. Adjust your font, font size, line spacing, letter spacing, text color, and background color. The live preview on the left updates as you make changes. When you're happy with your settings, click "Save and Start."

2

Open the popup to adjust settings at any time

Click the Pegasus View icon in the Chrome toolbar (top right of your browser) to open the extension popup. From here you can adjust any setting at any time.

3

Use the on/off toggle and reset button

Note the two key controls highlighted in the figure below:

Yellow Arrow

On/Off Toggle: the switch at the top right of the popup. Turns all customizations on or off without losing your saved preferences. Good for comparing against the original page.

Red Arrow

Reset to Defaults: the button at the very bottom of the popup. Sets every setting (font, size, spacing, colors) back to the original defaults. This cannot be undone.

Risks & Hazards

Safety Information

Pegasus View is a low-risk browser extension. Please review the following before use.

Not a medical device Pegasus View is designed for reading comfort and accessibility. It is not a substitute for professional medical, vision, or occupational therapy advice.
Layout changes Overriding fonts or colors on some websites may cause elements to overlap or shift. Use the on/off toggle to restore the original layout if needed.
Permissions The extension requests access to the active tab and local storage only. It does not collect or transmit any personal data or browsing history.
Photosensitivity Users sensitive to color changes should proceed carefully when selecting custom colors. High-contrast combinations may be uncomfortable.
Browser support Pegasus View is designed for Google Chrome and Microsoft Edge. Behavior in other Chromium-based browsers (Brave, Opera) is untested.
